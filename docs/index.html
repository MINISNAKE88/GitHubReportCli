GitHub Quick Reporter

A simple, fast terminal tool built to grab any GitHub user's profile summary in seconds—without having to click around the website.

Learn More

Why I Built This

When you want to see what someone has been working on on GitHub, you usually have to click through their profile, open their repositories tab, and look at each project one by one. It takes a lot of time.

I built this tool to solve that problem. You just type a username into the console, and it instantly prints out a clean summary of their top projects and coding statistics right inside your terminal screen.

What Information You Get

Top 5 Projects

Instantly shows the user's most popular repositories based on how many stars and forks they have earned from the community.

Favorite Languages

Looks at all their public work and counts up which coding languages they use the most (like C#, JavaScript, or Python).

Pending Code Reviews

Checks if there are any open projects where other developers are currently waiting for this user to review their code.
